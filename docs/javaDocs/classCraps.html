Craps ist ein Würfelspiel, wobei ein Spieler gegen die Bank spielt. Die Regeln sind die folgenden:

  1. Nur der Spieler würfelt. Es sind immer zwei sechs-seitige Würfel.
  2. Würfelt er eine Summe von 7 oder 11, gewinnt der Spieler.
  3. Würfelt er eine Summe von 2, 3 oder 12, verliert der Spieler.
  4. Jede andere Summe wird als "Point" bezeichnet und das Spiel fährt wie folgt fort:
    1. Würfelt der Spieler jetzt eine 7, verliert er.
    2. Würfelt er die selbe Summe, wie der "Point", gewinnt er.
    3. Es wird solange gewürfelt, bis eines der beiden Fälle eintritt.

Der "Craps-Test" ist zwei Test in einem, die Gewinnchance eines Spieles beträgt 244/495. Die Länge eines Spiels folgt einer geometrischen Verteilung, dabei werden Spiele, die länger als 20 Runden gehen, auf diese Anzahl reduziert.
