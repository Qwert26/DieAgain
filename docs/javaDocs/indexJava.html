test-Paket

Das Paket "test" enthält alle derzeit implementierten Tests.

util-Paket

Das Paket "util" enthält Unterstützungs-Klassen, die von der Testbatterie verwendet werden.

random-Paket

Das Paket "random" enthält zusätzliche Zufallsgeneratoren. Diese sind grundsätzlich "thread-safe" implementiert worden.

simulation-Paket

Das Paket "simulation" enthält Simulationen, welche man in neue Tests konvertieren könnte. Dies bringt immer gewisse Gefahren mitsich, denn wenn der verwendete Generator schlecht ist, wird die Simulation falsche Ergebnisse liefern. Falsche Ergebnise führen zu einem nicht-funktionalen Test.
