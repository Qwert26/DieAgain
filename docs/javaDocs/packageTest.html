Funktionale Tests

Diese Tests haben sich als korrekt implementiert herausgestellt in Bezug zu ihrem theoretischen Hintergrund.

  • Rank einer binärer Matrix
  • Bit-Verteilungs
  • Bit-Flip

    • mit Überlappen der Bytes

Eingeschränkte Tests

Diese Tests sind in einen oder mehreren Parametern eingeschränkt, diese können nicht geändert werden.

  • DNA
  • OPSO
  • OQSO
  • Gorilla

Nicht-Funktionale Tests

Diese Tests haben Probleme, welche sich nicht kurzer Zeit lösen lassen. Daher sollte man diese nicht verwenden.
